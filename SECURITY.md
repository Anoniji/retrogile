## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

### Preferred Method

Report vulnerabilities by emailing **[security@retrogile.com](mailto:security@retrogile.com)**

### What we will do

1. **Acknowledge** your report within 48 hours
2. **Analyze** the vulnerability within 7 days
3. **Release** a fix as soon as possible
4. **Credit** you in our Hall of Fame and release notes

### What to include

```
Title: [Brief summary]
Version: [tag/branch]
Description: [Detailed explanation]
Steps to reproduce: [Code/log/URL]
Impact: [CVSS score if possible]
```


### Expected Response Timeline

- **Initial Response**: 7 days
- **Analysis Complete**: 30 days
- **Patch Available**: Priority-based


## Security Scope

✅ Source code
✅ Dependencies
✅ Documentation
✅ Build process
❌ Third-party hosting infrastructure
❌ Design decisions / feature requests
